Keep your Asana token private

This page can remember an Asana personal access token only in this browser, on this site. Do not enable that option on a shared or untrusted computer, and never put a token in this HTML file or a Git repository. Google access is authorized in a Google pop-up and its short-lived token is kept only while this tab is open.
